# 🌿 ColorSite — Prodigy Web Dev Task 1

A clean, colorful, and fully responsive single-page website built as part of the **Prodigy InfoTech Web Development Internship (Task 1)**. Features a fixed navigation bar, smooth scrolling, scroll-triggered navbar styling, and a mobile-friendly hamburger menu.

## 🔗 Repository

[github.com/SanuKumarsingh/prodigy_wd_01](https://github.com/SanuKumarsingh/prodigy_wd_01)

## ✨ Features

- **Fixed Navbar** — Stays at the top while scrolling; switches to a dark theme after 50px scroll
- **Responsive Design** — Collapses into a hamburger menu on mobile screens (≤768px)
- **Smooth Scrolling** — Seamless anchor-based navigation between sections
- **Colorful Sections** — Each section has a distinct pastel background for visual clarity
- **FontAwesome Icons** — Lightweight icon support via CDN

## 📁 Project Structure

```
prodigy_wd_01/
└── task-1.html   # Single self-contained file (HTML + CSS + JS)
```

## 🚀 Getting Started

No build tools or dependencies required. Just open the file in a browser:

```bash
git clone https://github.com/SanuKumarsingh/prodigy_wd_01.git
cd prodigy_wd_01
open task-1.html
```
